# ⚔️ Batalla por la Tierra Media (Console Edition)

Motor de simulación de combate desarrollado en **Java** para profundizar en la lógica de programación y el uso de estructuras de datos avanzadas.

### 🕹️ Características
* **Simulación Automática:** Ejecución de batallas épicas basada en estadísticas y modificadores raciales sin intervención manual.
* **Lógica POO:** Aplicación de Herencia y Polimorfismo para gestionar las diferentes facciones (Héroes vs Bestias).
* **Gestión de Datos:** Uso de `ArrayList` e `Iterators` para procesar el flujo de combate y las bajas en tiempo real.

### 🚀 Ejecución
1. Importar el proyecto en Eclipse (o cualquier IDE Java).
2. Ejecutar la clase **`Aplicacion.java`**.
3. Seguir el log de la batalla a través de la consola del sistema.
